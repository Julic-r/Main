1) QR-Code scannen und Internetseite auf dem Mobilgerät laden lassen.
2) Sobald die Seite geladen ist, dient die blaue Fläche als Spielcontroller.
Blaue Fläche drücken um das Spiel zu laden.
3) Sobald das Spiel geladen ist, startet das nächste drücken der blauen Fläche das Spiel!
4) Steuere das HRW-Logo durch das drücken der blauen Fläche durch die Hindernisse.
